================================================================================
                   INVENTORY MANAGER — GAME PLAN
================================================================================

A Discord bot that accepts natural-language inventory input from users,
parses and categorizes the items, and outputs structured data to a Google
Sheet. Designed for small businesses using just a phone.

3. CATEGORIZATION APPROACH
================================================================================

Hybrid two-stage parsing:

Stage 1 — Rule-based (free, default):
  - Keyword maps defined in categories.py
  - Example: "laptop" -> Electronics, "chair" -> Furniture, "flour" -> Food
  - User can add custom categories via /add-cat command

Stage 2 — AI fallback (optional, ~$0.01/call):
  - If rule-based yields "Uncategorized", optionally call OpenAI/Claude
  - Disabled by default; toggle with /toggle-ai

WHY HYBRID: Zero cost for the common case, AI as a safety net for unusual or
ambiguous items. No recurring costs unless you enable the AI feature.

5. DISCORD BOT COMMANDS
================================================================================

Command     | Args         | Description
------------|--------------|-----------------------------------------------------
/inv        | <items...>   | "inventory 5 laptops, 10 mice, 3 monitors"
            |              |  -> parses, categorizes, writes to sheet, confirms
            |              |
/categories | —            | Lists all defined categories and their keywords
            |              |
/add-cat    | <name>       | "Office paper pens stapler"
            |  <keywords>  |  -> creates Office Supplies category
            |              |
/sheet      | —            | Returns a link to the Google Sheet
            |              |
/recent     | [n=5]        | Shows the last N entries from the sheet
            |              |
/toggle-ai  | —            | Enable or disable AI fallback categorization

6. PROJECT STRUCTURE
================================================================================

InventoryManager/
├── main.py                  # Entry point, wires everything together
├── config.py                # Loads .env, validates secrets
├── requirements.txt         # Python dependencies
├── .env.example             # Template for required env vars
├── bot/
│   ├── discord_bot.py       # Bot client setup, intents, lifecycle
│   └── handlers.py          # Slash command handlers
├── parser/
│   ├── extractor.py         # NLP: extract items & quantities from raw text
│   ├── categorizer.py       # Rule-based: map item names -> categories
│   ├── categories.py        # Default category definitions & keyword maps
│   └── ai_categorizer.py    # OpenAI/Claude fallback (optional)
└── sheets/
    ├── google_sheets.py     # Sheets auth, append, read operations
    └── schema.py            # Column definitions & formatting
